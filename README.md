Overview
==========

This project is an effort to combine existing community projects as well as new MongoDB related tools into a single, easily redistributable application.

Features:

* Database Visualizer
* Query Analyzer
* Log Explorer
* Database Explorer
* Query Interface
* Document Editor
* REST API
* Notification & Alerts
